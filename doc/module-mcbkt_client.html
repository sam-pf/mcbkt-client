Methods

(static) ajax_as_promise(url, methodopt, dataopt)

Makes an ajax call to url and returns a Promise object.
Parameters:
Name Type Attributes Default Description
url String URL to connect to.
method String <optional>
 "get" Http(s) method such as "get" or "post" (case-independent). If url does not use http(s) protocol, then this argument is ignored.
data JSON.stringfy-able <optional>
 Any data that will be sent as JSON along with the URL request. Any false value of this argument means no JSON data. Any true value will be converted to JSON to be sent. Naturally, such value also causes the "Content-Type" header to be set to "json" for the XMLHttpRequest object that opens request to url. Type requirement: JSON.stringify is called on data in order to prepare JSON to send and so the value of this argument should be JSON.stringify-able.

Returns:
A Promise object. The returned object can be used as shown in the example.
Example
ajax_as_promise ('https://...', ...)
  .then ((data) => {
     // do stuff with data
  })
  .catch ((err) => {
     // do stuff on error
  })
