(static) ajax_as_promise(url, methodopt, dataopt, headeropt)

Makes an ajax call to url and returns a Promise object.
Parameters:
Name Type Attributes Default Description
url String URL to connect to.
method String <optional>
 "get" Http(s) method such as "get" or "post" (case-independent). If url does not use http(s) protocol, then this argument is ignored.
data <optional>
 Any data that will be sent as JSON along with the URL request. Any false value of this argument means no JSON data. Any true value will be converted to JSON to be sent. Naturally, such value also causes the "Content-Type" header to be set to "json" for the XMLHttpRequest object that opens request to url.
header Object <optional>
 A hashmap that will be set as request header.

(static) post_logdata_for_mcbkt_analysis(logdataopt, urlopt)

Makes an ajax call to UKDE to post logdata and potentially get a MCBKT calculation done.
Parameters:
Name Type Attributes Description
logdata <optional>
 This must be the logdata in the form of a hashmap.
url String <optional>
 The URL for the MCBKT analysis. For security, no sensitive information must appear in the URL (or logdata). The current default value is a temporary one that should work "for a while". However, it may stop working at some point, e.g., if better proxy URLs are available.
Source:
  • mcbkt-client.js, line 117
Returns:
A Promise object. The returned object can be used as shown in the example. When the promise is successfully fulfilled, the returned data may not contain any MCBKT analysis result. Only when the accumulated data merit a new MCBKT analysis, data will contain MCBKT analysis result.
Example
ajax_as_promise ('https://...', ...)
  .then ((data) =>
     // do stuff with data
  )
  .catch ((err) =>
     // do stuff on error

(static) post_scores_for_mcbkt_analysis(dataopt, urlopt)

Makes an ajax call to UKDE to post score data and potentially get a MCBKT calculation done.
Parameters:
Name Type Attributes Description
data <optional>
 This must be a hashmap with an entry "scores", which must hold an array of numbers, normally bound within the [0,1] range. Optional entries include "score-max' and "times".
url String <optional>
 The URL for the MCBKT analysis. For security, no sensitive information must appear in the URL (or data). The current default value is a temporary one that should work "for a while". However, it may stop working at some point, e.g., if better proxy URLs are available.
Source:
  • mcbkt-client.js, line 87
Returns:
A Promise object. The returned object can be used as shown in the example. Just because the promise is successfully fulfilled does not mean that the returned data will contain MCBKT analysis result. If error field does not exist, data will contain valid MCBKT analysis result.
