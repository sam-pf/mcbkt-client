Module: mcbkt-client default export: post_logdata_for_mcbkt_analysis.

By design, the utility functions contained in this module are content-agnostic and merely make generic ajax calls. So, this module has no knowledge about how UKDE connections actually work. The actual work is to be carried out by the (hidden and hard-coded) URLs referenced in this file. Such work should be carried out in a secure manner in the backend: this is the job left for the users of this module.
